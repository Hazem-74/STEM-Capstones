# STEM Capstone Projects

STEM High School SGD (Sustainable Development Goals) Goals Projects

## Overview
This repository contains documentation, posters, and portfolios for student-led capstone projects aligned with the United Nations Sustainable Development Goals (SDGs). Each project demonstrates interdisciplinary learning in science, technology, engineering, and mathematics (STEM) to address real-world global challenges.
